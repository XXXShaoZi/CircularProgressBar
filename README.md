# CircularProgressBar
###### simple circular progress bar for android.
---

开发安卓项目时需要一个圆环形进度条，发现安卓没有原生支持，所以自己实现了一个。
###### 功能上需要支持：
1. 支持动态设定/获取当前进度
2. 支持使用XML参数定制控件
3. 支持动画效果
